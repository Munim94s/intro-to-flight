Maximum rate of climb of an aircraft is directly proportional to the difference of maximum power available from the engine and the power required by the airplane to overcome aerodynamic drag. The difference is called the excess power

There is no right way of designing an aircraft to achieve the goals, it's always an open-ended problem

However there are some general classes of airplane configuration:

- Conventional configuration - The configuration used by most modern aircraft. This configuration is a monoplane (Single set of wings) with a horizontal and vertical stabilizer at the back

- Canard configuration - Horizontal Stabilizer mounted ahead of the wings instead of the back. The horizontal stabilizer in this location is called the canard surface

Canard is a french word meaning "duck"

We can also classify an aircraft with it's set of wings:
- Monoplanes - Planes with single set of wings, most common today
- Biplanes - Planes with two sets of wings. Biplanes were popular in the earlier days as using two sets of short wings on top of each other provided enhanced structural strength compared to a single, longer span wing.

 As cantilevered wing design was introduced, it gradually became more accepted as the main technical reason for biplanes evaporated
